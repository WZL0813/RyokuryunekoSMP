指令列表

查看所有可以使用的指令

  • 只展示基础指令详细指令请进服尝试
指令 二级指令 描述
/velocity /velocity plugins 查看服务器安装的插件
/multilogin /multilogin link to <玩家ID> 将两个帐号合并为一个玩家
/tp /tp <玩家ID> 传送至某个玩家
/player /player <假人名字> <假人行为> 生成一个假人/使假人做某个行为（假人带有Bot_后缀）
/skin /skin set <正版玩家名字> 使用一个正版玩家的皮肤
/skin /skin url "https://网.站/皮肤.png" 使用一个自定义皮肤
